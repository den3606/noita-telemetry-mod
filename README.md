# Noita Telemetry MOD

Production MOD release mirror for [noita-telemetry](https://github.com/den3606/noita-telemetry).

## Install

1. Open [Releases](https://github.com/den3606/noita-telemetly-mod/releases/latest) and download the latest `.zip`.
2. Extract the `noita-telemetry` folder into your Noita `mods/` directory.
3. Enable **Unsafe mods** in the in-game mod menu.
4. Enable **Noita Telemetry** and restart Noita.

Cloud upload: create an API token on the dashboard **Settings** page and save it to
`mods/noita-telemetry/noita-telemetry.token` (one line).
